# 安装教程Install
## 环境要求
Windows 7及以上/Mac OS/Linux
Python环境3.7.3及以上
系统中安装Git
## 第一步 克隆存储库
### Outside Asia
`git clone https://github.com/FangcatChina/Easy-HTML.git`
### 国内推荐
`git clone https://gitee.com/FangcatChina/Easy-HTML.git`
## 第二步 检查Python版本
### Windows/Mac
`python`
### Linux
`python3`
## 第三步 安装Easygui
### Windows/Mac
`pip install easygui`
### Linux
`sudo pip3 install easygui`
## 第四步 运行
### Windows/Mac
```
cd Easy-HTML
python code.py
```
### Linux
```
cd ./Easy-HTML
python3 code.py
```
